运算符：算术、比较、逻辑、三元、短路运算

1. 算术运算符

// 算术运算符 console.log("1 + 2:", 1 + 2); // 3 console.log("5 - 3:", 5 - 3); // 2 console.log("2 * 4:", 2 * 4); // 8 console.log("10 / 2:", 10 / 2); // 5 console.log("10 % 3:", 10 % 3); // 1 // 使用变量来演示递增操作 let num1 = 1; console.log("++num1:", ++num1); // 2 let num2 = 1; console.log("num2++:", num2++); // 1 (先使用后自增) console.log("num2 自增后:", num2); // 2

2. 比较运算符

5. 短路运算

// 短路运算 const a1 = 0 || "默认值"; // 0 为假值，返回 "默认值" const b1 = "有值" || "默认值"; // "有值" 为真值，返回 "有值" const c1 = 1 && "后面的值"; // 1 为真值，返回 "后面的值" const d1 = 0 && "后面的值"; // 0 为假值，返回 0 console.log("a1:", a1); console.log("b1:", b1); console.log("c1:", c1); console.log("d1:", d1);
执行结果：

短路运算的原理：

  • || 运算符：如果第一个操作数为真值，则返回第一个操作数；否则返回第二个操作数
  • && 运算符：如果第一个操作数为假值，则返回第一个操作数；否则返回第二个操作数
  • 短路运算常用于设置默认值、条件执行等场景
