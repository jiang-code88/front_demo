5. do...while 循环

// do...while 循环 let k = 0; do { console.log("do...while 循环 k:", k); k++; } while (k < 3);
执行结果：

do...while 与 while 的区别：

  • do...while 循环至少执行一次循环体
  • while 循环只有在条件为真时才执行循环体

6. for...of 循环

// for...of 循环 (遍历可迭代对象) const arr2 = [1, 2, 3]; for (const item of arr2) { console.log("for...of 循环 item:", item); }
执行结果：

for...of 循环的特点：

  • 用于遍历可迭代对象（数组、字符串、Map、Set 等）
  • 直接获取值，而不是索引
  • 不能直接遍历对象的属性

7. for...in 循环

// for...in 循环 (遍历对象属性) const obj3 = { name: "张三", age: 20 }; for (const key in obj3) { console.log("for...in 循环 key:", key, "value:", obj3[key]); }
执行结果：

for...in 循环的特点：

  • 用于遍历对象的可枚举属性
  • 获取的是属性名，需要通过属性名访问属性值
  • 会遍历原型链上的属性
