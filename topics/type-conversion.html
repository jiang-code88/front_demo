显式与隐式类型转换

1. 显式类型转换

// 显式类型转换 console.log("Number('123'):", Number('123')); // 123 console.log("String(123):", String(123)); // "123" console.log("Boolean(0):", Boolean(0)); // false console.log("Boolean(''):", Boolean('')); // false console.log("Boolean(null):", Boolean(null)); // false console.log("Boolean(undefined):", Boolean(undefined)); // false console.log("Boolean(NaN):", Boolean(NaN)); // false console.log("Boolean(1):", Boolean(1)); // true console.log("Boolean('Hello'):", Boolean('Hello')); // true
执行结果：

显式类型转换的方法：

  • Number()：转换为数字类型
  • String()：转换为字符串类型
  • Boolean()：转换为布尔类型
  • parseInt()：解析字符串为整数
  • parseFloat()：解析字符串为浮点数
  • toString()：对象转换为字符串

2. 隐式类型转换

// 隐式类型转换 console.log("'1' + 2:", '1' + 2); // "12" (字符串拼接) console.log("1 + '2':", 1 + '2'); // "12" (字符串拼接) console.log("1 - '2':", 1 - '2'); // -1 (减法会转换为数字) console.log("'123' * 2:", '123' * 2); // 246 (乘法会转换为数字) console.log("+'123':", +'123'); // 123 (一元加号转换为数字) console.log("!'Hello':", !'Hello'); // false (非运算转换为布尔值) console.log("!!'Hello':", !!'Hello'); // true (双重非运算转换为布尔值)
执行结果：

隐式类型转换的场景：

  • 字符串拼接：当使用 + 运算符且其中一个操作数是字符串时，会将另一个操作数转换为字符串
  • 算术运算：当使用算术运算符时，会将操作数转换为数字
  • 比较运算：会根据需要转换操作数的类型
  • 逻辑运算：会将操作数转换为布尔值
  • 一元运算符：+ 会转换为数字，! 会转换为布尔值

3. 真值与假值

JavaScript 中的假值：

  • false
  • 0 和 -0
  • '' (空字符串)
  • null
  • undefined
  • NaN

JavaScript 中的真值：

  • 除了上述假值以外的所有值
  • 包括非空字符串、非零数字、对象、数组等
