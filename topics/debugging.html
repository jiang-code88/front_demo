1. console 对象的基本方法

// console 对象 console.log("console.log: 普通日志"); console.warn("console.warn: 警告日志"); console.error("console.error: 错误日志"); console.info("console.info: 信息日志");

3. console.group() 方法

// console.group console.group("分组开始"); console.log("分组内的日志 1"); console.log("分组内的日志 2"); console.groupEnd();
执行结果：
提示：请打开浏览器的开发者工具（F12）查看控制台输出的分组

4. debugger 语句

// debugger 语句 (执行到这里会暂停) // debugger;

debugger 语句的作用：

  • 当执行到 debugger 语句时，浏览器会暂停代码执行
  • 可以在暂停时查看变量值、调用栈等信息
  • 需要在浏览器的开发者工具中使用
  • 注释掉的 debugger 语句不会执行

5. 其他常用的 console 方法

常用的 console 方法：

  • console.clear()：清空控制台
  • console.time(label)：开始计时
  • console.timeEnd(label)：结束计时并输出时间
  • console.trace()：输出调用栈
  • console.assert(condition, message)：当条件为假时输出错误信息
