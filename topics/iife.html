立即调用函数表达式（IIFE）

1. 基本 IIFE

// 基本 IIFE (function() { const privateVar = "私有变量"; console.log("IIFE 内部:", privateVar); })(); // console.log(privateVar); // 会报错，privateVar 是 IIFE 内部的私有变量
执行结果：

IIFE 的基本语法：

  • (function() { ... })();
  • 第一个括号将函数声明转换为函数表达式
  • 第二个括号立即调用这个函数

2. IIFE 带参数

// IIFE 带参数 (function(name) { console.log("IIFE 带参数:", `Hello, ${name}!`); })("张三");

3. IIFE 返回值

// IIFE 返回值 const result = (function(a, b) { return a + b; })(10, 20); console.log("IIFE 返回值:", result); // 30

4. IIFE 的实际应用

IIFE 的常见应用：

  • 创建私有作用域：避免变量污染全局作用域
  • 模块化：实现早期的模块模式
  • 立即执行代码：需要立即执行的初始化代码
  • 防止变量提升：控制变量的作用域

5. IIFE 与闭包的关系

IIFE 与闭包的关系：

  • IIFE 可以创建一个私有作用域
  • 在 IIFE 内部定义的函数可以形成闭包
  • IIFE 常与闭包结合使用，实现模块化和数据封装
