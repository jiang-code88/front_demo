4. 箭头函数与 this 指向

// 箭头函数与 this 指向 const obj4 = { name: "张三", sayName1: function() { console.log("函数声明 this.name:", this.name); // 张三 }, sayName2: () => { console.log("箭头函数 this.name:", this.name); // undefined (指向全局对象) } }; obj4.sayName1(); obj4.sayName2();
执行结果：

箭头函数的 this 指向特点：

  • 箭头函数没有自己的 this，它会继承外层作用域的 this
  • 箭头函数的 this 在定义时就确定了，不会在调用时改变
  • 箭头函数不能作为构造函数使用
  • 箭头函数没有 arguments 对象

5. 三种函数的区别对比

构造函数 可以 可以 不可以
简写形式 无 无 有（单参数、单表达式）
