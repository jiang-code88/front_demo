参数：默认值、剩余参数、arguments 对象

1. 默认参数

// 默认参数 function greet(name = "陌生人") { console.log("默认参数 greet():", `Hello, ${name}!`); } greet(); // Hello, 陌生人! greet("张三"); // Hello, 张三!
执行结果：

默认参数的特点：

  • 当函数调用时没有提供参数，会使用默认值
  • 默认值可以是表达式、函数调用等
  • 默认参数的计算是惰性的，只有在需要时才会计算

2. 剩余参数

// 剩余参数 function sum(...numbers) { return numbers.reduce((total, num) => total + num, 0); } console.log("剩余参数 sum(1, 2, 3, 4):", sum(1, 2, 3, 4)); // 10
执行结果：

剩余参数的特点：

  • 使用 ... 语法收集剩余的参数
  • 返回一个数组，包含所有剩余的参数
  • 只能作为函数的最后一个参数

3. arguments 对象

// arguments 对象 (仅在函数声明和函数表达式中可用) function showArgs() { console.log("arguments 对象:", arguments); console.log("arguments[0]:", arguments[0]); } showArgs(1, "hello", true);
执行结果：

arguments 对象的特点：

  • 是一个类数组对象，包含函数调用时的所有参数
  • 具有 length 属性，但没有数组的方法
  • 仅在函数声明和函数表达式中可用，箭头函数中不可用
  • 可以通过索引访问参数，如 arguments[0]

4. 剩余参数与 arguments 的区别

剩余参数与 arguments 的主要区别：

  • 剩余参数是一个真正的数组，具有数组的所有方法
  • arguments 是一个类数组对象，需要转换为数组才能使用数组方法
  • 剩余参数只包含那些没有对应形参的实参，而 arguments 包含所有实参
  • 箭头函数中没有 arguments 对象，但可以使用剩余参数
